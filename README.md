# Trabajo práctico de bucles de JavaScript con DOM

Este es el tercer trabajo práctico de funciones y arrays de JavaScript utilizando el DOM para manipular elementos HTML.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- DOM

## Instrucciones de uso

Para utilizar este trabajo práctico, simplemente sigue estos pasos:

1. Descarga o clona el repositorio en tu ordenador o haciendo click [aqui](https://silviojrcc.github.io/tercer-trabajo-js/)
2. Abre el archivo index.html en tu navegador web
3. Haz clic en los botones para ver cómo funcionan las diferentes acciones
